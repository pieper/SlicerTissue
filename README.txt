This is a port to 3D Slicer of code that dates back to my (Steve Pieper's) PhD
thesis at the MIT Media Lab completed in 1991.

More details are available:

http://www.isomics.com/caps

Pieper, S. "CAPS: Computer-Aided Plastic Surgery", Ph.D. thesis, Massachusetts Institute of Technology, 1991.

Pieper, S., Rosen, J., & Zeltzer, D. (1992), "Interactive graphics for plastic surgery: A task-level analysis and implementation". Proceedings of the 1992 Symposium on 3D Graphics, (pp. 127-134). Cambridge, MA; ACM Press.

Pieper, S., Laub, D., Rosen, J., (1995), "A Finite Element Model for Simulating Plastic Surgery", in Plastic and Reconstructive Surgery, Vol. 96, No. 5 (pp. 1100-1105).

The code is made available for public review, but use of the code requires permission from Steve Pieper.
